1.ВЗУ (внешние запоминающие устройства) в информатике предназначены для долговременного хранения информации.
Назначение Информация, записанная на ВЗУ, доступна для использования даже при выключенном компьютере. В отличие от оперативной памяти, внешняя не имеет прямой связи с процессором.
Применение ВЗУ используются для: Постоянного хранения программ и данных в компьютерах. Переноса данных между компьютерами с помощью сменных носителей (например, дискет). Архивации и резервного копирования данных.
Принцип работы Работа ВЗУ включает два процесса: запись и чтение информации. Запись происходит с помощью записывающей головки, которая изменяет состояние носителя (например, намагничивает участки диска или создаёт углубления на поверхности оптического диска). Чтение осуществляется считывающей головкой, которая определяет моменты, когда при движении носителя под ней оказываются границы между участками с противоположными состояниями намагниченности.
